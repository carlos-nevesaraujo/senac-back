# Variaveis

Variaveis servem para guardar dados na memoria do programa.

Exemplos:
- nome de uma pessoa
- idade
- texto de uma tarefa
- resultado de uma consulta

Em JavaScript, uma variavel pode ser criada com `const`, `let` ou `var`.

- Use `const` quando o valor nao vai ser reatribudo.
- Use `let` quando o valor pode mudar depois.
- Evite `var` em projetos novos.

Exemplo:

```js
const nome = 'Ana';
let idade = 20;
idade = 21;
```

Na pratica, variaveis ajudam a deixar o codigo mais legivel e evitam repetir valores varias vezes.
