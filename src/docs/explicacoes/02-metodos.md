# Metodos

Metodo e uma funcao que faz parte de um objeto ou de uma classe.

Na linguagem do dia a dia, muita gente usa "metodo" e "funcao" quase como sinonimos.

Exemplos:
- `res.json()` envia resposta em JSON.
- `String.trim()` remove espacos do inicio e do fim.
- `Array.includes()` verifica se um valor existe no array.

Exemplo:

```js
const texto = '  Curso SENAC  ';
const limpo = texto.trim();
```

Neste projeto, varios metodos sao usados para trabalhar com banco, texto, resposta HTTP e autenticacao.
