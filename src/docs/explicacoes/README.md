# Explicacoes basicas

Esta pasta junta conceitos essenciais para quem esta começando em programacao e APIs.

## Ordem de leitura

1. `01-variaveis.md` - o que sao variaveis e como elas guardam valores.
2. `02-metodos.md` - o que sao metodos e funcoes.
3. `03-api.md` - o que e uma API e para que ela serve.
4. `04-metodos-http.md` - tipos de requisicao HTTP mais usados.
5. `05-status-codes.md` - o significado dos principais status code.

Leia na ordem se estiver estudando do zero: variaveis -> metodos -> API -> requisicoes -> status code.
